# API de Location de Véhicules
Description

Cette API permet de gérer une flotte de véhicules pour un service de location.
Elle offre des opérations CRUD complètes pour gérer les utilisateurs, véhicules et réservations.

L’API est documentée avec OpenAPI 3.0 et un fichier openapi.yaml est fourni pour consultation via Swagger UI.

Technologies utilisées

- PHP 8
- Composer 
- MongoDB 
- Swagger

Installation

Cloner le dépôt :

git clone https://github.com/ton-utilisateur/location-vehicules-api.git

cd location-vehicules-api


Installer les dépendances avec Composer :

composer install

Lancer le serveur PHP :

php -S localhost:8000

Accéder à la documentation technique :
Visiter : http://localhost:8000/swagger.html
